# Navbar

Narbar 模块用于创建顶部导航栏。

## 用法

| 类名                | 描述                   |
| ------------------- | ---------------------- |
| `.fu-navbar-top`    | 创建一个顶部导航栏     |
| `.fu-navbar-bottom` | 创建一个底部导航栏     |
| `.fu-navbar-item`   | 创建一个导航栏的子项目 |
| `.fu-navbar-link`   | 创建一个导航链接       |

```html
<div class="fu-navbar-top">
  <div class="fu-navbar-item"></div>
  <a href="" class="fu-navbar-item fu-navbar-link"></a>
</div>
```
