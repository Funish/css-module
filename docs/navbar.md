# Navbar

Narbar 模块用于创建顶部导航栏。

## 用法

| 类名              | 描述                       |
| ----------------- | -------------------------- |
| `.fu-navbar`      | 创建一个导航栏容器         |
| `.fu-navbar-item` | 创建一个导航栏容器的子项目 |
| `.fu-navbar-link` | 创建一个导航链接           |

```html
<div class="fu-navbar">
  <div class="fu-navbar-item"></div>
  <div class="fu-navbar-item">
    <a href="" class="fu-navbar-link"></a>
  </div>
</div>
```
