# 关于

Funish CSS 是 [Thus.Fun](https://thus.fun) 团队开发的 `Funish` 项目的一部分，作为 Web 前端界面的开发构件之一，以 `CSS` 这样简单的形式存在，旨在利用纯 `CSS` 代码的方式实现 `Funish` 前端的设计理念。
