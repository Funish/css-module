# Pagination

Pagination 模块用于创建分页导航

## 用法

使用类选择器 `.fun-pagination` 以应用样式。

```html
<div class="fun-pagination">
  <span>Previous</span>
  <em>1</em>
  <a href="#url">2</a>
  <a href="#url">3</a>
  <span>…</span>
  <a href="#url">8</a>
  <a href="#url">9</a>
  <a href="#url">10</a>
  <a>Next</a>
</div>
```
