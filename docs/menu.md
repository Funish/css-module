# Menu

Menu 模块用于创建菜单。

## 用法

通过应用类选择器 `.fun-menu-*` 创建菜单。

| 类名               | 描述           |
| ------------------ | -------------- |
| `.fun-menu`        | 创建菜单容器   |
| `.fun-menu-header` | 创建菜单标题   |
| `.fun-menu-item`   | 创建菜单子项目 |

```html
<div class="fun-menu">
  <div class="fun-menu-header">Header</div>
  <li class="fun-menu-item">item1</li>
  <li class="fun-menu-item">item2</li>
  <li class="fun-menu-item">item3</li>
</div>
```
